# 地图图片文件夹

## 使用说明

请将以下地图图片放入此文件夹：

| 文件名                 | 用途              | 说明                   |
| ---------------------- | ----------------- | ---------------------- |
| `ali-loop-overview.jpg` | 阿里大环线总览图  | 查看完整的阿里大环线路线 |
| `kailash-kora-map.jpg`  | 冈仁波齐转山路线图 | 转山三天路线、住宿点     |
| `day03-lhasa-shigatse.jpg` | Day 3 路线图     | 拉萨→羊湖→日喀则       |
| `day05-saga-darchen.jpg`   | Day 5 路线图     | 萨嘎→玛旁雍错→塔钦     |
| `day07-kora-d1.jpg`     | Day 7 转山 D1     | 塔钦→止热寺路线         |
| `day08-kora-d2.jpg`     | Day 8 转山 D2     | 卓玛拉山口路线          |
| `day09-kora-d3.jpg`     | Day 9 转山 D3     | 祖楚寺→塔钦路线         |

## 图片要求

- 格式：JPG 或 PNG
- 建议宽度：750–1200px（手机端合适）
- 文件大小：每张 < 500KB（微信加载友好）
- 可先放占位图/截图，出发前替换为高清版

## 来源建议

1. 高德/百度地图截图
2. 两步路/六只脚户外轨迹截图
3. OpenStreetMap 截图（阿里地区）
4. Google Earth 截图
5. 自制路线标注图

## 重要提示

⚠️ 出发前务必长按图片保存到手机相册，确保无网络时也可查看！
